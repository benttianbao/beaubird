查询 eBird 区域记录

输入 eBird API 密钥后，可查询某个国家 / 地区 / 县区 / 位置代码下的最近观测。 这些区域结果只用于查看，不会写入你的个人记录。

区域代码可填国家、州省、区县或 locId。示例：CN、CN-31、 US-CA、L7884500。 API 密钥仅保存在当前浏览器的本地存储中。

本工具聚焦区域查询；如果你只想快速看一个地区最近有哪些鸟，这里直接查询即可。

浙江当季可能出现鸟种

固定分析 CN-ZJ，按目标日期前后窗口统计多年历史记录，并用最近 30 天记录标记是否已确认。

BirdReport 代理查询

通过代理请求 BirdReport，按省 / 市 / 区、观测地点和时间范围查看符合条件的鸟种。网页版可连接本地代理，Android APK 版可直接使用内置代理。

网页版需要先在本机运行代理脚本；Android APK 版会自动启用应用内代理。

未解锁鸟种查询

输入记录用户姓名，按浙江 588 种鸟种名录核对该用户还缺哪些鸟种。缺口列表按浙江历史记录数从多到少排列，地点按报告编号顺序快速加载。

浙江稀有鸟种监测

基线规则固定为“仅浙江省、不限日期、记录次数小于等于 500”。名单会保存在当前浏览器中；页面保持打开时，会每隔 1 小时自动检查一次所选日期的浙江数据。
