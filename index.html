eBird

查询 eBird 区域记录

使用 eBird API Key 查询区域最近观测，并分析浙江当季可能出现的鸟种。

区域代码可填国家、州省、区县或 locId。示例：CN、CN-31、 US-CA、L7884500。

API 密钥仅保存在当前浏览器的本地存储中，区域结果只用于查看。

等待 eBird 区域记录 填入 API Key、区域代码和回溯天数后，最近观测会显示在这里。

浙江当季可能出现鸟种

固定分析 CN-33，按目标日期前后窗口统计多年历史记录，并用最近 30 天记录标记是否已确认。

等待当季分析 选择目标日期并开始分析后，会列出浙江当季候选鸟种。

BirdReport

BirdReport 查询

按省、市、区、观测地点和时间范围查看符合条件的鸟种。

等待 BirdReport 查询 选择时间和地区后，符合条件的鸟种结果会显示在这里。
